# Audio Transcription Frontend

A React SPA app that detects live speech from audio input, and transcribes it into text.

## Features
- Text highlighting
- Start playing from certain words
- Local endpoint support (uses localhost:5001 by default)

## Prerequisites
- nodejs

## Installation

1. install dependancies:

```bash
npm install
```

## Usage

### Start the Server

```bash
npm start
```

The local server will be available at `http://localhost:3000`
